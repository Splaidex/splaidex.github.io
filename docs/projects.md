# Проекты

## 🎮 SteamParser
Парсер изображений из Steam Community по названию игры.

- Поиск игры прямо внутри приложения
- Подключение через Steam API
- Создание папок для каждой игры
- Без лимитов на выгрузку
- GUI с анимациями на PyQt6

---

## 📄 JsonGenerator
Генератор JSON-файлов из CSV с автодокументацией.

- Загрузка CSV — на выходе готовый JSON
- Уникальные ключи при каждой генерации
- Поддержка массивов
- Автоматически создаёт `.md` с расшифровкой ключей
- Кастомное имя файла через GUI

---

## ✂️ WMCut
Инструмент для удаления вотермарок с видео.

- Превью видео прямо в приложении
- Выделение зоны мышкой
- Мультивыбор зон очистки
- GUI на PyQt6
